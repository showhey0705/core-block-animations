Add smooth entrance animations to core blocks with fade, slide, zoom, and rotate effects.

== Description ==

Core Block Animations is a lightweight WordPress plugin that adds beautiful entrance animations to your WordPress core blocks. Enhance your site's visual appeal without affecting performance or requiring coding knowledge.

**Key Features:**

* **Multiple Animation Types**: Choose from fade, slide, zoom, and rotate effects
* **Customizable Timing**: Control animation duration and delay
* **Easing Options**: Select from various easing functions for natural motion
* **Performance Optimized**: Lightweight code that doesn't slow down your site
* **Easy to Use**: Simple controls integrated into the block editor
* **Core Blocks Support**: Works with all standard WordPress blocks
* **Mobile Friendly**: Responsive animations that work on all devices

**Animation Types Available:**

* **Fade**: Smooth fade-in effects
* **Slide**: Slide in from any direction (up, down, left, right)
* **Zoom**: Scale-in animations
* **Rotate**: Rotation-based entrance effects

**Perfect For:**

* Landing pages
* Portfolio sites
* Business websites
* Blog posts
* Marketing pages

The plugin integrates seamlessly with the WordPress block editor (Gutenberg), adding animation controls to the block inspector panel. No additional setup required - simply install, activate, and start animating your blocks!

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/core-block-animations` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Open any post or page in the block editor.
4. Select any core block and find the "Animation" panel in the block inspector.
5. Choose your preferred animation type, timing, and easing options.

== Frequently Asked Questions ==

= Which blocks are supported? =

Core Block Animations works with all standard WordPress core blocks including paragraphs, headings, images, columns, groups, and more.

= Does this affect page loading speed? =

No, the plugin is designed to be lightweight and performance-optimized. Animations only load when needed and don't impact your site's speed.

= Can I customize the animation timing? =

Yes, you can control both the duration of the animation and the delay before it starts through the block inspector panel.

= Will animations work on mobile devices? =

Absolutely! All animations are responsive and work smoothly across desktop, tablet, and mobile devices.

= Do I need coding knowledge to use this plugin? =

Not at all! The plugin provides an intuitive interface within the WordPress block editor. Simply select a block and choose your animation preferences.

= Can I disable animations on specific blocks? =

Yes, animations are applied on a per-block basis. You can choose which blocks to animate and which to leave static.
